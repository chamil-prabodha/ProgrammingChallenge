
Installation, configuring and running the program

1)To install TicTacToe, run the set up file.
2)Open the icon created on your desktop to play the game.
3)Make sure you are connected to the internet.
Note: The game might be a bit slow depending on the internet connection.

Operating Instructions

1)To view player scores, click on "View" and sub-item "Scores" on the Tool Strip.
2)To exit the game, click on "File" and sub-item "Exit"on the Tool Strip.
